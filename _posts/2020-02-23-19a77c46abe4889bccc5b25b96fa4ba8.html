1360. Number of Days Between Two Dates #Easy

题目描述

Write a program to count the number of days between two dates.

The two dates are given as strings, their format is YYYY-MM-DD as shown in the examples.

Example 1:

Input: date1 = "2019-06-29", date2 = "2019-06-30"
Output: 1

Example 2:

Input: date1 = "2020-01-15", date2 = "2019-12-31"
Output: 15

Constraints:

  • The given dates are valid dates between the years 1971 and 2100.

题目解析

首先，根据日期格式把较小日期date3和较大日期date4的年、月、日分别提取出来；其次，模拟增加较小日期，注意在d1增加过程中，需要根据不同月份进行进位，在m1增加过程中，需要进行进位；最后，直到两个日期完全重合，返回计数结果res。

C++ 实现

class Solution {
public:
    int daysBetweenDates(string date1, string date2) {
        string date3 = min(date1, date2);
        string date4 = max(date1, date2);
        int y1 = stoi(date3.substr(0, 4));
        int m1 = stoi(date3.substr(5, 2));
        int d1 = stoi(date3.substr(8, 2));
        int y2 = stoi(date4.substr(0, 4));
        int m2 = stoi(date4.substr(5, 2));
        int d2 = stoi(date4.substr(8, 2));
        int res = 0;
        while (y1 != y2 || m1 != m2 || d1 != d2) {
            ++res;
            ++d1;
            switch (m1) {
                case 1: case 3: case 5: case 7:
                case 8: case 10: case 12:
                    if (d1 == 32) {
                        ++m1;
                        d1 = 1;
                    }
                    break;
                case 4: case 6: case 9: case 11:
                    if (d1 == 31) {
                        ++m1;
                        d1 = 1;
                    }
                    break;
                case 2:
                    if ((y1 % 4 == 0 && y1 % 100 != 0) || y1 % 400 == 0) {
                        if (d1 == 30) {
                            ++m1;
                            d1 = 1;
                        }
                    } else {
                        if (d1 == 29) {
                            ++m1;
                            d1 = 1;
                        }
                    }
                    break;
            }
            if (m1 == 13) {
                ++y1;
                m1 = 1;
            }
        }
        return res;
    }
};

1361. Validate Binary Tree Nodes #Medium

题目描述

You have n binary tree nodes numbered from 0 to n - 1 where node i has two children leftChild[i] and rightChild[i], return true if and only if all the given nodes form exactly one valid binary tree.

If node i has no left child then leftChild[i] will equal -1, similarly for the right child.

Note that the nodes have no values and that we only use the node numbers in this problem.

Example 1:

Input: n = 4, leftChild = [1,-1,3,-1], rightChild = [2,-1,-1,-1]
Output: true

Example 2:

Input: n = 4, leftChild = [1,-1,3,-1], rightChild = [2,3,-1,-1]
Output: false

Example 3:

Input: n = 2, leftChild = [1,0], rightChild = [-1,-1]
Output: false

Example 4:

Input: n = 6, leftChild = [1,-1,-1,4,-1,-1], rightChild = [2,-1,-1,5,-1,-1]
Output: false

Constraints:

  • 1 <= n <= 10^4
  • leftChild.length == rightChild.length == n
  • -1 <= leftChild[i], rightChild[i] <= n - 1

题目解析

首先，声明一个计数数组cnt，用于存放每个节点的访问次数；其次，遍历节点及其左右子节点，同时进行计数；然后，遍历cnt，统计其中访问次数为1的节点个数为res，同时如果发现某个节点的访问次数大于2，说明该节点同时为多个节点的子节点，不符合二叉树的要求，返回false；最后，如果res的值不为1，说明存在多个根节点，返回false，否则返回true。

C++ 实现

class Solution {
public:
    bool validateBinaryTreeNodes(int n, vector<int>& leftChild, vector<int>& rightChild) {
        vector<int> cnt(n, 0);
        for (int i = 0; i < n; ++i) {
            ++cnt[i];
            if (leftChild[i] != -1) ++cnt[leftChild[i]];
            if (rightChild[i] != -1) ++cnt[rightChild[i]];
        }
        int res = 0;
        for (int i = 0; i < n; ++i) {
            if (cnt[i] == 1) ++res;
            else if (cnt[i] > 2) return false;
        }
        return res == 1;
    }
};

1362. Closest Divisors #Medium

题目描述

Given an integer num, find the closest two integers in absolute difference whose product equals num + 1 or num + 2.

Return the two integers in any order.

Example 1:

Input: num = 8
Output: [3,3]
Explanation: For num + 1 = 9, the closest divisors are 3 & 3, for num + 2 = 10, the closest divisors are 2 & 5, hence 3 & 3 is chosen.

Example 2:

Input: num = 123
Output: [5,25]

Example 3:

Input: num = 999
Output: [40,25]

Constraints:

  • 1 <= num <= 10^9

题目解析

计算num+2的平方根并向上取整为fact，从fact开始递减遍历，寻找绝对值之差最小的两个因数即可。

C++ 实现

class Solution {
public:
    vector<int> closestDivisors(int num) {
        int fact = sqrt(num+2) + 1, gap = INT_MAX;
        array<int, 2> arr = {num+1, num+2};
        vector<int> res;
        for (int i = fact; i > 0; --i) {
            for (int j = 0; j < 2; ++j) {
                if (arr[j] % i == 0 && abs(i - arr[j]/i) < gap) {
                    res = {i, arr[j]/i};
                    gap = abs(i - arr[j]/i);
                }
            }
        }
        return res;
    }
};

1363. Largest Multiple of Three #Hard

题目描述

Given an integer array of digits, return the largest multiple of three that can be formed by concatenating some of the given digits in any order.

Since the answer may not fit in an integer data type, return the answer as a string.

If there is no answer return an empty string.

Example 1:

Input: digits = [8,1,9]
Output: "981"

Example 2:

Input: digits = [8,6,7,1,0]
Output: "8760"

Example 3:

Input: digits = [1]
Output: ""

Example 4:

Input: digits = [0,0,0,0,0,0]
Output: "0"

Constraints:

  • 1 <= digits.length <= 10^4
  • 0 <= digits[i] <= 9
  • The returning answer must not contain unnecessary leading zeros.

题目解析

首先，将digits按从大到小的顺序进行排序，声明类型为array<string, 3>的arr和tmp，其中第i个元素以字符串形式表示除3余i的最大数字，声明flag表示digits中是否存在0；其次，遍历digits，将遍历到的num添加至arr中第j个数字的末尾就是arr[j]+to_string(num)，形成一个新的数字，其除3的余数就是(j+num)%3，于是使用arr[(j+num)%3]和arr[j]+to_string(num)中的较大值更新tmp[(j+num)%3]；然后，如果arr[num%3]为空，则用num进行更新；最后，3的最大倍数自然存放在arr[0]中，如果arr[0]为空且digits中存在0，返回"0"，否则返回arr[0]。

C++ 实现

class Solution {
public:
    string largestMultipleOfThree(vector<int>& digits) {
        sort(digits.rbegin(), digits.rend());
        array<string, 3> arr, tmp;
        bool flag = false;
        for (int i = 0; i < digits.size(); ++i) {
            int num = digits[i];
            tmp = arr;
            for (int j = 0; j < 3; ++j) {
                if (arr[j] != "") {
                    string &s1 = tmp[(j+num)%3];
                    string s2 = arr[(j+num)%3];
                    string s3 = arr[j] + to_string(num);
                    if (s2.size() == s3.size())
                        s1 = max(s2, s3);
                    else
                        s1 = (s2.size() > s3.size()) ? s2 : s3;
                }
            }
            arr = tmp;
            if (num == 0) flag = true;
            if (arr[num%3] == "" && num != 0) arr[num%3] = to_string(num);
        }
        if (arr[0] == "" && flag)
            return "0";
        else
            return arr[0];
    }
};
